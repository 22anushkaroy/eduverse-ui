# EduVerse Login & Signup UI

A simple Flutter UI project for an educational mobile application called EduVerse.

## Features
- Login screen
- Signup screen
- Google sign-in button UI
- Student/Teacher role selection
- Responsive design
- Reusable widgets
- Clean and modern educational theme

## Technologies Used
- Flutter
- Dart

## Project Structure
- main.dart
- login_screen.dart
- signup_screen.dart
- reusable widgets
